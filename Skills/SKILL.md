# RetroPrinciples

**复盘知识库 Skill**：查询核心原则、获取最佳实践建议、查阅历史案例。

## 使用方式

```
/principles "关键词"      # 搜索原则
/checklist               # 获取启动前检查清单
/example "问题类型"       # 查看相关案例
/reminder "场景"         # 获取场景提醒
```

## 功能

| 功能 | 命令 | 说明 |
|------|------|------|
| 搜索原则 | `/principles "用户"` | 搜索包含"用户"的原则 |
| 检查清单 | `/checklist` | 项目启动前检查清单 |
| 案例查询 | `/example "成本估算"` | 查看相关案例 |
| 场景提醒 | `/reminder "技术选型"` | 获取技术选型提醒 |
| 列出所有 | `/principles list` | 列出所有原则 |

## 10 条核心原则

1. **用户需求验证** - 没问过用户 = 猜
2. **ROI 保守主义** - 乐观是幻想，保守是智慧
3. **XY Problem 警惕** - 问 5 次"为什么"再动手
4. **多代理分析 ROI** - 花小钱防大祸
5. **竞品沉默** - 90% 有原因
6. **文档金字塔** - 5% 结论，20% 方案，75% 细节
7. **隐性成本** - 算到的成本只是冰山
8. **可测量** - 模糊 = 无效
9. **技术选型** - 先 POC 再上车
10. **分阶段决策** - 每个阶段必须有退出点
